# 一键装机指南（华为 HBN-AL80）

> push 到 `master` 后，GitHub Actions 自动构建 APK 并上传阿里云 OSS 固定链接。
> 本文档指导**新会话**把最新构建装到真机。新开对话时把本文件贴给 Claude 即可照做。

| 项 | 值 |

## 前提检查

```powershell
adb version          # 确认 adb 可用
adb devices          # 应列出一台 device（不是 unauthorized/offline）
```

- 手机：**开发者选项 → USB 调试** 打开；数据线连电脑，USB 模式选「传输文件」
- 首次装机手机会弹「**是否允许通过 USB 安装应用**」→ 点允许
- `adb devices` 若显示 `unauthorized`：看手机弹窗点「允许此电脑调试」

- 状态是 `completed / success` 才继续（一次构建约 **14 分钟**）

## 第 2 步：一键装机（在项目根目录 `d:\my-project\soloist-carbin` 跑）

```powershell
$url = "https://soloist-apk.oss-cn-guangzhou.aliyuncs.com/apk/latest.tar.gz"
Invoke-WebRequest $url -OutFile latest.tar.gz
tar -xzf latest.tar.gz
$apk = Get-ChildItem *.apk | Sort-Object LastWriteTime -Descending | Select-Object -First 1
Write-Host "安装: $($apk.Name)"
adb install -r $apk.FullName
```

`-r` = 覆盖安装并保留数据（签到记录、账本、设置都在）。看到 `Success` 即成功。

## 常见问题

| 现象 | 处理 |
|------|------|
| `adb devices` 为空 | 换数据线 / 重插 / 手机确认调试弹窗 / `adb kill-server; adb start-server` |
| `INSTALL_FAILED_UPDATE_INCOMPATIBLE` | 签名不一致，需先卸载：`adb uninstall org.soloist.soloistcarbin`（**会清空数据**），再重装 |
| `INSTALL_FAILED_USER_RESTRICTED` | 华为拦截，手机上手动点「继续安装/仍要安装」，或关闭「纯净模式」 |
| 下载到的还是旧包 | CI `completed` 后 OSS 才覆盖，等 1–2 分钟重新下载 |
| 装完闪退 | `adb logcat -s python` 抓崩溃日志排查 |

## 第 3 步：验证「拍摄日」新功能（真机走一遍）

1. **上午上班前**进签到页 → 顶部出现「今天是拍摄日吗？」→ 点「设为拍摄日」
2. 三个时段卡消失 → 出现 **小猫庆祝动画** + 三按钮：完成拍摄 / 拍张现场 / 取消
3. 点「**拍张现场**」→ 调系统相机拍一张 → 提示「现场照已记录」
4. 点「**完成拍摄**」→ 复盘弹窗填 4 项（拍了什么/在哪/顺不顺/感想）→ 提交
5. 提交后 → **小狗摘星星庆祝浮层**（拍摄复盘已记录，奖励已入账）
6. 卡片变「拍摄复盘已完成」→ 点「查看战报」
7. 战报应含：日期卡三格标「拍摄」+ **拍摄复盘卡** + **一张大幅现场照**（刚才拍的）
